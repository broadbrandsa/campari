# Project Overview

## Campari Group Africa — Digital Advertising & Influencer Guidance Proposal

A standalone proposal website prepared by **BroadBrand** for **Campari Group Africa**, March 2026.

### Purpose

Present a comprehensive digital advertising and influencer guidance strategy for Campari Group Africa's portfolio across key African markets. The site consolidates regulatory research, platform-specific rules, and actionable recommendations into a single interactive document.

### Target Audience

- Campari Group Africa marketing teams
- Campari Group Africa compliance teams

### Tech Stack

| Layer       | Technology                        |
| ----------- | --------------------------------- |
| Framework   | Next.js (App Router)              |
| Language    | TypeScript                        |
| Styling     | Tailwind CSS v4                   |
| Components  | shadcn UI                         |
| Deployment  | Vercel (static Next.js export)    |

### Deployment

The site is deployed as a static Next.js export on Vercel, requiring no server-side runtime or database.
