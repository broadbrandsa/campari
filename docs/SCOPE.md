# Scope

## In Scope

### Content Sections

The proposal site comprises 7 content sections:

1. **Overview** — High-level strategy summary
2. **Platform Rules** — Advertising policies per platform
3. **Market Detail** — Country-by-country regulatory breakdown
4. **Kenya Meta Fix** — Specific remediation steps for Kenya Meta advertising
5. **Influencer Strategy** — Influencer engagement and compliance guidance
6. **90-Day Roadmap** — Phased activation plan
7. **Market Readiness** — Readiness assessment per market

### Interactive Elements

- Tabbed interface for platform-specific details
- Accordion components for market breakdowns
- Phase-selectable roadmap navigation

### Launch Milestones

| Phase   | Description      |
| ------- | ---------------- |
| Phase 1 | Build and deploy |
| Phase 2 | Client review    |
| Phase 3 | Iteration        |

## Out of Scope

- CMS integration
- Authentication or user accounts
- Multi-language support
- Payment processing
- API endpoints
